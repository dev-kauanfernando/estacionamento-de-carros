# 🚗 Sistema de Estacionamento em Java

Este é um projeto simples desenvolvido em Java para simular o funcionamento de um estacionamento.

---

## 📌 Funcionalidades

- ✅ Estacionar carros  
- ❌ Evitar placas duplicadas  
- 🔍 Buscar carro por placa  
- 🧾 Remover carro e calcular valor a pagar  
- 📋 Listar carros estacionados  

---

## 🧠 Lógica aplicada

O sistema utiliza:

- Vetores (`array`) para armazenar os carros  
- Programação orientada a objetos (POO)  
- Métodos auxiliares para reaproveitamento de código  

---

## 💰 Regra de cobrança

O valor é calculado com base no tempo estacionado:

Valor = tempo (horas) × R$5

---

## 📁 Estrutura do projeto

Carro.java  
Estacionamento.java  
Main.java  

---

## ▶️ Como executar

1. Compile os arquivos:

javac *.java

2. Execute o programa:

java Main
